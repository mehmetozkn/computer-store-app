# computer_store_app

Computer Store App

### Summary

- This project provides to add product to basket for existing users

### Tech Stack

- State Management with Provider and Mobx
- English and Turkish language support with Easy Localization
- MVVM Design Pattern
- REST API
- HTTP requests with Dio
